# Ireland-House-Price-Prediction
Using machine learning models to predict property purchase intent and prices in the Dublin, Ireland area.

## Project Overview
Analyzing the Irish Dublin property dataset to explore key factors influencing home buying decisions (location, size, price, energy rating, etc.) and build predictive models:
- **Purchase Intent Prediction**: Binary classification (buy or not buy)
- **House Price Prediction**: Regression (price per square foot)

## Tech Stack
- Python + pandas + scikit-learn
- Multiple machine learning models (Logistic Regression, Random Forest, Gradient Boosting, SVM, etc.)
- Complete data preprocessing pipeline (missing value handling, outlier detection, feature engineering)
